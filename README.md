# Tesla Clone Website

A responsive Tesla landing page clone built using only HTML and CSS.

## 🚀 Features

- Fully responsive design
- Tesla-inspired UI
- Smooth layout and sections
- Clean and simple code structure
- Pure HTML & CSS (No frameworks)

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📁 Project Structure

```bash
Tesla-Clone/
│── index.html
│── style.css
│── images/
│── videos/
```

## 📸 Preview

This project recreates the Tesla homepage design for learning and practice purposes.

## 🎯 Purpose

This project was built to improve frontend development skills using only HTML and CSS.
